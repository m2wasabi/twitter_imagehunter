# Twitter Imagehunter

### Requires
+ php
+ php-pecl-oauth
+ php-curl (php-common)
+ Account of Twitter, and your app

### Usage
1. Edit `config.php` to fill below.
  + CONSUMER_KEY
  + CONSUMER_SECRET
  + ACCESS_TOKEN
  + ACCESS_TOKEN_SECRET
2. Use from command line

```
php twitter_imagehunter.php <twitter_screen_name>
```

```
php twitter_imagehunter_s.php <some_keyword>
```
